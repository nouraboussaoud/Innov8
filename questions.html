1. Definition of a Process

A process (Process in English) is a program in execution. Every process has its own characteristics: an identification number, resources it uses such as open files, and is always in a state (running, waiting, etc.).

A process consists of:

  • An executable code of the running program.
  • A context that provides the environment of the process.

2. Characteristics of a Process

a. Static Characteristics:

  • A unique PID (Process Identifier).
  • An owner determining the process's access rights to resources: opening files, etc.
  • A parent process from which it inherits most characteristics.

b. Dynamic Characteristics:

  • Priority, execution environment.
  • Amount of resources consumed (CPU time used, etc.).
